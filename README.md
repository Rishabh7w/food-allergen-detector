# Food Allergen Detection System

A web-based AI-powered system that detects potential food allergens from images, ingredient lists, and food labels.

## Features

- **Multimodal Allergen Detection** — Analyze food photos, ingredient labels, and text-based ingredient lists
- **Personalized Allergy Profiles** — Create and manage multiple user allergy profiles
- **Risk Assessment Engine** — Quantitative risk scoring based on detected allergens and user sensitivity
- **Explainable AI Visualization** — Grad-CAM style attention heatmaps showing model focus regions
- **Scan History Tracking** — Review past analyses and results
- **Multi-User Profile Support** — Switch between profiles for different family members

## Technologies Used

- React
- TypeScript
- Supabase
- AI Vision Models (Gemini)
- Tailwind CSS
- Framer Motion
